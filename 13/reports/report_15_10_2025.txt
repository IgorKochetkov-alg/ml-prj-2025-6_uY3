Написан класс Нейронной сети с использованием декораторов на основе функций из библиотеки tf:
    
    - Составлена преварительная архитектура Нейронной сети:
        1-ый слой: два перцептрона [t,x] с масштабом 
        2-6 слои: по 10 перцептронов с ф-ей активации гиперпобилческим тангенсом
        7 слой: 1 выходной перцептрон со значением предсказываемой функции [u]
    
    - Для обучения использован оптимизатор adam + ошибка по MSE

    - Ошибка по MSE из: ошибка по Н.У. + ошибка по Г.У. + ошибка уравнения ( базово оно равно нулю -> посчитанное значение и будет ошибкой )

    - Заранее реализован класс логгирования для отслеживания ошибки по эпохам ( использовал аналогичный в одном из проектов )

    - Вылезает ошибка по None при взятии градиента, который нужен для полной производной в диффуре... ^_^
